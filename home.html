Sistemas operacionais

O que são?

Como dito por Linus Torvald no documentário Revolution OS: "Sistemas operacionais não devem ser percebidos, sua única função e ser um intermédio entre o hardware e os programas usados pelo usuário". Sistemas operacionais facilitam o acesso à alguns recursos do computador, como memória ram, armazenamento e processamento.

Quais são sistemas operacionais?

Linux
Baseado no Unix, é um sistema operacional de código aberto que surgiu
Windows
Sistema proprietário da Microsoft, o primeiro lançamento foi em 1985
macOS
Sistema proprietário da Apple para seus computadores, lançado em 2001
Android
Sistema feito para dispositivos móveis a partir da modificação do Linux
iOS
Sistema da apple para o iPhone

Todos eles são de uso geral, mas cada um é melhor para determinados fins, fizemos uma tabela de comparação entre eles, apontando pontos negativos e positivos.

Os mais usados no mundo

Uma pesquisa feita pela StatCounter com dados refentes a julho de 2022 mostra quais são os sistema mais utilizados no mundo:

  1. Android: 44,66%
  2. Windows: 27,92%
  3. iOS: 17,85%
  4. macOS: 5,37%
  5. Outros: 2,29%
  6. Linux: 1,02%

Distribuições Linux

O Linux nasceu com o intuito de ser livre e open source, ou seja, para que qualquer um pudesse acessar seu código, alterá-lo e depois distribuí-lo novamente, devido à isso possui inúmeras distribuições. Entre elas temos:

  • Linux Mint
  • Linux Ubuntu
  • Linux Debian
  • Kali Linux
  • Manjaro Linux
